# Solución
el problema me pedia encontrar 2 numeros compuestos que sumado den N, entonces cuando N sea par solo le resto 8 e imprimia 8 tmb (siempre da compuestos) y cuando N es impar hacia lo mismo pero con 9

numeros compuestos de un par = N - 8 y 8
numeros compuestos de un impar = N - 9 y 9
