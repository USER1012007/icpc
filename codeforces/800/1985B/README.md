# Solución
el problema me pedia el numero menor o igual a N que la sumatoria con ese multiplo de el mayor numero posible, entonces solo compare si es mayor a 3, y si si, imprimia 2, si no, imprimia N, esto por que la sumatoria de 2 hasta en N de += 2, era la sumatoria mayor posible.
