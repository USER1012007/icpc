# Solución
Solucion de complejidad constante, la "raíz digital¨ de un número es congruente con su modulo 9, para encontrar el k-th número con una de las 9 posibles
raíces digitales (1-9) hay que sumar $9 \cdot (k-1)$ a una raíz digital.
