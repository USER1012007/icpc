# Problema
[1695B](https://codeforces.com/problemset/problem/1695/B)

# Tags
Games, greedy

# Solución
Mike siempre gana si n es impar, en cambio si n es par, pierde quien tenga la pila con menor numero de piedras (solo se pueden mover entre sus pilas y jugando
óptimo siempre decidiran solo quitar una piedra, por tanto, quien tenga la pila con menor número de piedras pierde)
